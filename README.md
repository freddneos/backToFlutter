# Simple TODO List in Flutter

Simple TODO list in flutter , just to remember the basics of the framework.

- Data persistence with `path_provider`
- `ListView` with `Dismissible` Items to remove from list.

### Widgets

- Dismissible
- InputText
- Expanded
- RaisedButton
- RefreshIndicator
- ListView
- Icon
- Text
- Column
- Container
- Row
